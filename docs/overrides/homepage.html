Opteryx

Query your data, where it lives.

Get Started Star on Github

Opteryx is a SQL Query Engine with cross-platform, command-line and embedded Python skills.

Features

Multi-platform

Runs most places you can run Python; GCP, AWS, Azure, Kubernetes, Knative, your own servers or a Raspberry Pi.

In Place Analytics

Run queries against data in without loading or migrating into another system.

Simple

Use SQL to query your data, no learning curve for analysts.

Elastic & Available

Scale down to zero, and scale up to respond to thousands of concurrent queries within seconds.

Shared nothing, each query runs in its own environment.

Time Travel

Query data as at a point in time in the past to replay decision algorithms against facts as they were known in the past.

Open Source & Extensible

Fork, contribute, and review. Opteryx is available to all.

Use Cases

Using SQL to query data written by another process, such as logs
As a command line tool - bring the power and flexibility of SQL to filter and transform files
As an embeddable engine - a low-cost option to allow hundreds of analysts to each have part-time databases

Negative Use Cases

Don't use Opteryx for accounting systems, Opteryx is designed for bulk analysis of data.
Don't use Opteryx for end-customer facing systems, Opteryx is a low-cost solution, not a high-speed solution.
